# Python数据模型

> ⚠️ 不成熟的抽象和过早的优化一样，都会坏事。

Python语言的核心架构：

- 迭代
- 集合类
- 属性访问
- 运算符重载
- 函数和方法的调用
- 对象的创建和销毁
- 字符串表示形式和格式化
- 管理上下文（即`with`块）

**可变和不可变对象**

> 对象的字符串表示形式：交互式控制台和调试程序用`repr`函数获取字符串表示形式，且尽可能表达出如何用代码创建出被
> 打印的对象；`print`打印函数用`str`函数获取对终端用户友好的字符串表示形式。当一个对象没有实现`__str__`函数，
> 而python解释器又需要调用它的时候，解释器会用`__repr__`作为替代。

默认情况下，自定义类的实例的布尔值总被认为是真的，除非这个类自定义了`__bool__`或`__len__`特殊方法。
